<hr />

**`id`** : adalah sebuah atribut yang berfungsi untuk memberi id-name pada elemen html yang nantinya akan di panggil oleh CSS ataupun JAVASCRIPT, pengguna tidak akan melihat perubahan pada elemen yang di berikan atribut ini.

:::info CATATAN 

  - Pseudo ini berfungsi pada semua elemen.

  - Membutuhkan value dengan cara penulisan bebas sesuai yang kita inginkan dengan syarat tidak boleh mengandung spasi. 

:::

```html 
<h3 id="text-judul">Judul Conten</h3>
<p id="text-content">Lorem, ipsum dolor sit amet consectetur adipisicing elit...</p>
```

:::danger LARANGAN 

  1 elemen html tidak boleh memiliki 2 id seperti berikut ini :

  ```html 
  <p id="text-content color-blue">Hello World</p>
  ```

:::
