# HTML ATRIBUT

Atribut adalah bagian dari elemen HTML yang memberikan informasi tambahan tentang elemen tersebut. Atribut digunakan untuk mengontrol perilaku, tampilan, dan fitur-fitur lain dari elemen HTML. Setiap elemen HTML dapat memiliki satu atau lebih atribut, yang didefinisikan dalam tag elemen.

- Atribut di tuliskan di dalam tag pembuka seperti berikut :

```html 
<h1 atribut>Halo Dunia!</h1>
```

- Ada atribut yang memiliki nilai/value ada juga yang tidak. 

```html 
<h1 atribut="nilai-atribut">Halo Dunia!</h1>
```
