**`class`** : adalah sebuah atribut yang berfungsi untuk memberi class-name pada elemen html yang nantinya akan di panggil oleh CSS ataupun JAVASCRIPT, pengguna tclassak akan melihat perubahan pada elemen yang di berikan atribut ini.

:::info CATATAN 

  - Pseudo ini berfungsi pada semua elemen.

  - Membutuhkan value dengan cara penulisan bebas sesuai yang kita inginkan.

  - 1 elemen bisa saja memiliki lebih dari 1 nilai atau class-name dengan cara memberikan spasi. 

:::

- class dengan 1 nilai :

```html 
<h3 class="text-judul">Judul Conten</h3>
<p class="text-content">Lorem, ipsum dolor sit amet consectetur adipisicing elit...</p>
```

- class dengan lebih dari 1 nilai :

```html
<p class="center color-blue font-50">Nama saya adalah...</p>
```
