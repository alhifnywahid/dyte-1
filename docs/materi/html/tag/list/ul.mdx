<hr />

tag `<ul>` atau Unordered List befungsi untuk membuat list yang tidak ter - urut.

tag `<ul>` membutuhkan tag `<li>` di dalamnya agar menjadi sebuah list dengan tab.

### Atribut

- type

### Contoh

```html
<ul>
  <li>Es Teh</li>
  <li>Es Cendol</li>
  <li>Es Dawet</li>
</ul>
```

cara mengubah listnya bisa menggunakan atribut type seperti berikut ini :

```html
<ul type="circle/disc/square">
  <li>Es Teh</li>
  <li>Es Cendol</li>
  <li>Es Dawet</li>
</ul>
```
