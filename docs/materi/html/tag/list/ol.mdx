<hr />

tag `<ol>` atau Ordered List befungsi untuk membuat list yang ter - urut.

tag `<ol>` membutuhkan tag `<li>` di dalamnya agar menjadi sebuah list dengan tab.

### Atribut

- type

### Contoh 

```html 
<ol>
  <li>Es Teh</li>
  <li>Es Cendol</li>
  <li>Es Dawet</li>
</ol>
```

cara mengubah listnya bisa menggunakan atribut type seperti berikut ini : 

```html 
<ol type="1/a/A/i/I">
  <li>Es Teh</li>
  <li>Es Cendol</li>
  <li>Es Dawet</li>
</ol>
```
