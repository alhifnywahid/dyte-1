## Fungsi

Berfungsi untuk membuat link.

## Atribut

- **href** : untuk menyisipkan link.
- **target** : memberi petunjuk kepada browser bagaimana tautan tersebut harus dibuka ketika diklik oleh pengguna. Ada beberapa nilai yang dapat digunakan untuk atribut target:
  - **_blank **: Ketika tautan diklik, halaman yang ditautkan akan dibuka dalam jendela atau tab browser baru.
  - **_self**: Ini adalah perilaku default. Halaman yang ditautkan akan terbuka di jendela atau tab yang sama tempat tautan tersebut berada.
  - **_parent**: Jika halaman memiliki bingkai (frames), tautan akan dibuka di dalam bingkai induk dari bingkai saat ini.
  - **_top**: Jika halaman memiliki bingkai (frames), tautan akan membuka halaman di luar semua bingkai, menggantikan seluruh halaman saat ini.
