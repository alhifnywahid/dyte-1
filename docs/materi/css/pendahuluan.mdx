## syntax

```css
selector {
  properti: value;
}
```

## penjelasan

- `selector` : Selector adalah kata kunci untuk memilih elemen HTML berdasarkan elemen, class dan id.

- `kurung kurawal { dan }` : Blok deklarasi adalah tempat untuk menulis properti beserta valuenya, kita harus menuliskan properti dan value di dalam blok kodenya.

- `properti` : Properti adalah sekumpulan syntax style yang akan terrapka pada elemen yang dipilih, seperti warna, warna background, ukuran font, dll.

- `titik dua :` Adalah sebuah simbol yang harus di tulisk sebelum mengisi value pada properti.

- `value` : Untuk memberi nilai pada propertinya, jadi setiap properti pasti mempunyai nilai, seperti warna: ungu, biru, merah..

- `titik koma ;` : Setiap properti harus di akhiri dengan simbol titik koma ; karena agar terbaca oleh program ketika mengisi lebih dari 1 properti di dalam blok deklarasi.

## contoh

```css
h1 {
  color: red;
}
```

- kodingan di atas berarti akan memberikan warna merah pada elemen h1.
