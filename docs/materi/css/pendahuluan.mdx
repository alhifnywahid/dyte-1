## syntax

```css
selector {
  properti: value;
}
```

## penjelasan

`selector` : Selektor adalah kata kunci untuk memilih elemen HTML berdasarkan elemen, class dan id.

`{ dan }` : Blok deklarasi adalah tempat untuk menulist atribut dan properti css.

`properti` : Properti adalah sekumpulan syntax yang akan diberikan kepada elemen yang dipilih. cotoh : color, padding, font dll.

`value` : Untuk memberi nilai pada propertinya.

`; titik koma` : Setiap properti harus di akhiri dengan tanda titik koma ;

## contoh

```css
h1 {
  color: red;
}
```

kodingan di atas berarti akan memberikan warna merah pada elemen html h1.
