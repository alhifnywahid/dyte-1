`display` adalah sebuah properti yang mengontrol bagaimana sebuah elemen HTML ditampilkan di halaman web. Properti `display` memungkinkan kita untuk mengubah tata letak dan perilaku elemen-elemen HTML.

Pada dasarnya semua elemen html di bungkus dalam sebuah kotak dan mempunyai display default yang berbeda, biasanya elemen pada html mempunyai display default `inline` atau `block`.

properti display sendiri mempunyai beberapa value di antaranya :

- [inline](#1-inline)
- [block](#2-block)
- [inline-block](#3-inline-block)
- [flex](#4-flex)
- [grid](#5-grid)
- [none](#6-none)

### 1. inline

Elemen yang memiliki property display: inline; akan mengisi tempat yang kosong di sebelah elemen jika masih ada tempat (tidak membuat baris baru jika masih ada tempat kosong). Kita tidak dapat mengubah atau mengatur nilai lebar dan tinggi dari elemen tersebut. Selain itu, nilai margin-top dan margin-bottom juga tidak berpengaruh pada elemen inline. Dari situlah lebar dan panjang kotak pada elemen akan mengikuti isi konten - nya.

contoh elemen yang mempunyai default display inline di antaranya : `<span>`, `<a>`, `<strong>`, `<label>`, `label`, dan masih banyak lagi.
