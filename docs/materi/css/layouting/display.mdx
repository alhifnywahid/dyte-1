`display` adalah sebuah properti yang mengontrol bagaimana sebuah elemen HTML ditampilkan di halaman web. Properti `display` memungkinkan kita untuk mengubah tata letak dan perilaku elemen-elemen HTML.

Pada dasarnya semua elemen html di bungkus dalam sebuah kotak dan mempunyai display default yang berbeda, biasanya elemen pada html mempunyai display default `inline` atau `block`.

properti display sendiri mempunyai beberapa value di antaranya :

- [inline](#1-inline)
- [block](#2-block)
- [inline-block](#3-inline-block)
- [flex](#4-flex)
- [grid](#5-grid)
- [none](#6-none)
