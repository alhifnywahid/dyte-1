## Menggunakan Tag HTML

Berfungsi memilih elemen html berdasarkan tagnya.

### Contoh 

- Tuliskan nama tag html yang ingin di ubah style nya.

```css 
h1 {
  color: green;
}
```

### Penjelasan

Contoh di atas akan memilih semua elemen html yang menggunakan tag **h1**.

## Menggunakan Class

Berfungsi untuk memilih elemen html berdasarkan class.

### Contoh 

- Tuliskan symbol titik (.) lalu tuliskan nama class-nya.

```css
.text-header {
  font-size: 40px;
}
```

### Penjelasan

Contoh di atas akan memilih elemen yang mempunyai class **text-header** dan akan di ubah ukuran fontnya menjadi 40px.


## Menggunakan Id

Berfungsi untuk memilih elemen html berdasarkan id.

### Contoh 

- Tuliskan symbol pagar (#) lalu tuliskan nama id-nya.

```css
#text-footer {
  text-align: center;
}
```

### Penjelasan

Contoh di atas akan memilih elemen yang mempunyai id **text-footer** dan akan di ubah poisisnya menjadi ke tengah.
