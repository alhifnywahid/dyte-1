<hr />

**`cursor`** adalah sebuah property yang berfungsi untuk mengontrol tampilan kursor ketika pengguna berinteraksi dengan elemen tertentu di halaman web. Properti ini memungkinkan pengembang web untuk menentukan jenis kursor yang akan ditampilkan ketika pengguna meletakkan kursor mouse di atas suatu elemen.

### - Value 

Berikut adalah beberapa nilai yang dapat digunakan pada properti cursor dalam CSS beserta fungsinya:

| Value            | Deskripsi                                                                                             |
|------------------|-------------------------------------------------------------------------------------------------------|
| auto             | Browser akan menentukan jenis kursor yang sesuai berdasarkan konteks. Ini adalah nilai default.       |
| default          | Menampilkan kursor default untuk elemen tersebut (biasanya kursor panah).                             |
| none             | Tidak menampilkan kursor sama sekali, membuat kursor tidak terlihat.                                  |
| context-menu     | Menampilkan kursor konteks menu, menunjukkan bahwa elemen tersebut menyediakan menu konteks.          |
| help             | Menampilkan kursor tanda tanya, menunjukkan bahwa elemen tersebut memberikan bantuan atau informasi tambahan. |
| pointer          | Menampilkan kursor tangan, menunjukkan bahwa elemen tersebut dapat diklik (seperti tautan atau tombol). |
| progress         | Menampilkan kursor menunggu, menunjukkan bahwa pengguna harus menunggu proses tertentu sebelum dapat melanjutkan. |
| wait             | Menampilkan kursor menunggu, serupa dengan progress, menunjukkan bahwa pengguna harus menunggu proses tertentu sebelum dapat melanjutkan. |
| cell             | Menampilkan kursor sel, menunjukkan bahwa elemen tersebut adalah sel tabel yang dapat dipilih.       |
| crosshair        | Menampilkan kursor berbentuk salib, biasanya digunakan untuk menunjukkan mode pemilihan atau pemberian. |
| text             | Menampilkan kursor isian teks, menunjukkan bahwa elemen tersebut dapat menerima input teks.         |
| vertical-text    | Menampilkan kursor isian teks vertikal, digunakan untuk teks vertikal.                                |
| alias            | Menampilkan kursor alias, menunjukkan bahwa elemen tersebut adalah tautan yang sudah dikunjungi sebelumnya. |
| copy             | Menampilkan kursor menyalin, menunjukkan bahwa elemen tersebut dapat disalin.                         |
| move             | Menampilkan kursor tanda arah panah yang menunjukkan bahwa elemen tersebut dapat digeser (seperti gambar atau elemen dengan properti position yang diatur). |
| no-drop          | Menampilkan kursor tanpa tindakan drop, menunjukkan bahwa suatu tindakan drop tidak diizinkan pada elemen tersebut. |
| not-allowed      | Menampilkan kursor larangan, menunjukkan bahwa tindakan tertentu tidak diizinkan.                    |
| grab             | Menampilkan kursor tanda genggam, menunjukkan bahwa elemen tersebut dapat ditarik (dragged).         |
| grabbing         | Menampilkan kursor tanda genggam tergenggam, menunjukkan bahwa elemen tersebut sedang ditarik (dragged). |
| ew-resize        | Menampilkan kursor tanda panah horisontal ganda, menunjukkan bahwa elemen tersebut dapat diubah ukurannya secara horizontal. |
| ns-resize        | Menampilkan kursor tanda panah vertikal ganda, menunjukkan bahwa elemen tersebut dapat diubah ukurannya secara vertikal. |
| nesw-resize      | Menampilkan kursor tanda panah diagonal, menunjukkan bahwa elemen tersebut dapat diubah ukurannya diagonal dari kiri atas ke kanan bawah. |
| nwse-resize      | Menampilkan kursor tanda panah diagonal, menunjukkan bahwa elemen tersebut dapat diubah ukurannya diagonal dari kanan atas ke kiri bawah. |
| col-resize       | Menampilkan kursor tanda panah horizontal, menunjukkan bahwa elemen tersebut dapat diubah lebarnya. |
| row-resize       | Menampilkan kursor tanda panah vertikal, menunjukkan bahwa elemen tersebut dapat diubah tingginya. |
| zoom-in          | Menampilkan kursor tanda tambah, menunjukkan bahwa elemen tersebut dapat diperbesar (zoom in).    |
| zoom-out         | Menampilkan kursor tanda kurang, menunjukkan bahwa elemen tersebut dapat diperkecil (zoom out).   |
| all-scroll       | Menampilkan kursor tanda panah ke segala arah, menunjukkan bahwa elemen tersebut dapat digulirkan di semua arah. |
| vertical-resize  | Menampilkan kursor vertikal resize.                                                                  |
| horizontal-resize| Menampilkan kursor horizontal resize.                                                               |
| none             | Tidak menampilkan kursor sama sekali.                                                                |

- all-scroll: Menampilkan kursor tanda panah ke segala arah, menunjukkan bahwa elemen tersebut dapat digulirkan di semua arah.

- vertical-resize: Menampilkan kursor vertikal resize.

- horizontal-resize: Menampilkan kursor horizontal resize.

- none: Tidak menampilkan kursor sama sekali.
