**`position`** adalah sebuah property CSS yang digunakan untuk menentukan bagaimana elemen ditempatkan dalam dokumen. Property ini memungkinkan elemen untuk ditempatkan secara tepat dan dapat digunakan untuk membuat layout yang kompleks dan responsif.

property position sendiri memiliki 5 value yang biasa di gunakan :

- static
- relative
- absolute
- fixed
- sticky

  <TabItem label="relative" value="relative" default>

relative adalah value yang memungkinkan elemen untuk ditempatkan relatif terhadap posisi normalnya. Dengan menggunakan `position: relative;`, kita dapat mengatur posisi elemen dengan properti `top`, `right`, `bottom`, dan `left` relatif terhadap posisinya yang sebelumnya, dimana elemennya seakan maju 1 dimensi ke atas tetapi posisi sebelumnya masih ada.

<Codepen dataSlugHash="vYMVOEv" title="property - position : relative" />

`top: 10px;` : bergeser dari atas posisi elemen semula ke bawah 10pixel.
`left: 10px;` : bergeser dari kiri posisi elemen semula ke kanan 10pixel.

property ini biasanya hanya di gunakan untuk memajukan sebuah elemen tidak untuk menggeser geser sebuah elemen.

absolute adalah value yang memungkinkan elemen untuk ditempatkan mutlak terhadap posisi normalnya. Dengan menggunakan `position: absolute;`, kita dapat mengatur posisi elemen dengan properti `top`, `right`, `bottom`, dan `left` mutlak terhadap posisinya yang sebelumnya, dimana elemennya seakan dipindahkan ke posisi absolut tanpa memperdulikan posisi sebelumnya.
