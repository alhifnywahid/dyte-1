**`:hover`** adalah sebuah pseudo-class yang berfungsi untuk menerapkan style pada elemen ketika kursor berada di atas elemen yang telah di tentukan.

:::info CATATAN 
  - pseudo ini berfungsi pada semua elemen.
:::

#### Contoh

<Tabs>
<TabItem value="DEMO" label="DEMO" default>

<Codepen dataSlugHash="xxBvbRx" title="pseudo class - hover" />

</TabItem>

<TabItem value="COPY CODE" label="COPY CODE">

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>:hover</title>
    <style>
      h1:hover {
        color: green;
      }
    </style>
  </head>
  <body>
    <h1>heading 1</h1>
    <p>paragraf</p>
  </body>
</html>
```

</TabItem>
</Tabs>

#### Penjelasan

Contoh di atas akan menerapkan style pada elemen `h1` dimana ketika kursor berada di atas elemen `h1` maka warnanya akan berubah menjadi hijau.
