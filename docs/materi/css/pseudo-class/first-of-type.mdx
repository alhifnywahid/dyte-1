**`:first-of-type`** adalah sebuah pseudo-class yang berfungsi untuk memilih elemen child pertama dari parent - nya. Jika ada elemen yang berbeda tipe/tagnya maka akan di pilih semua di setiap child pertama dari setiap tipe elemen tersebut.

:::info CATATAN

- pseudo ini berfungsi pada semua elemen.

:::

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>:first-of-type</title>
    <style>
      /* contoh 1 */
      .div1 span:first-of-type {
        font-weight: bold;
        color: red;
      }

      /* contoh 2 */
      .div2 :first-of-type {
        font-weight: bold;
        color: red;
      }
    </style>
  </head>
  <body>
    <!-- contoh 1 -->
    <div class="div1">
      <p>pragraph ke 1</p>
      <span>in span ke 1</span>
      <p>pragraph ke 2</p>
      <span>in span ke 2</span>
      <p>pragraph ke 3</p>
      <span>in span ke 3</span>
    </div>

    <hr>

    <!-- contoh 2 -->
    <div class="div2">
      <h3>heading 1</h3>
      <span>span 1</span>
      <p>paragraf 1</p>
      <h3>heading 2</h3>
      <span>span 2</span>
      <p>paragraf 2</p>
    </div>
  </body>
</html>
```

</TabItem>
</Tabs>

#### Penjelasan

- Contoh 1 : elemen `<span> ke 1` akan di pilih karena merupakan elemen child pertama dengan tipe `<span>` dari class `div1`
- Contoh 2 : elemen `heading 1, span 1` dan `paragraf 1` akan dipilih karena merupakan elemen child yang berbeda tipe dan posisinya merupakan child pertama dari elemen class `div2`
