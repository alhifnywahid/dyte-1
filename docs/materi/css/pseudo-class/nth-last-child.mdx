<hr />

**`:nth-last-child()`** fungsi-nya sama dengan pseudo :nth-child(), perbedaan-nya hanya pada pemulaian pada urutan elemen dimana :nth-last-child() ini akan di mulai dari elemen child terakhir dari induk atau parent-nya.

- 1. :nth-last-child(n)
- 2. :nth-last-child(A)
- 3. :nth-last-child(An)
- 4. :nth-last-child(An+B)
- 5. :nth-last-child(An-B)
- 6. :nth-last-child(odd)
- 7. :nth-last-child(even)
- 8. :nth-last-child(-n+B)

Untuk penjelasan argumen-nya silahkan cek di [`:nth-child(arg)`](./nth-child.mdx)

:::info CATATAN

- pseudo ini berfungsi pada semua elemen.

:::

## 1. :nth-last-child(n)

berfungsi untuk memilih beberapa elemen tertentu ataupun semua elemen yang berada di dalam parentnya.

<Tabs>
<TabItem value="DEMO" label="DEMO" default>

<Codepen dataSlugHash="MWRWdVd" title="pseudo class - nth-last-child(n)" />

</TabItem>

<TabItem value="COPY CODE" label="COPY CODE">

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>:nth-last-child(n)</title>
    <style>
      .main1 :nth-last-child(n) {
        color: red;
      }
    </style>
  </head>
  <body>
    <div class="main1">
      <p>paragraf 1</p>
      <p>paragraf 2</p>
      <p>paragraf 3</p>
      <div>
        <span>elemen span</span>
      </div>
    </div>

    <div class="main2">
      <p>paragraf 1</p>
      <p>paragraf 2</p>
      <p>paragraf 3</p>
    </div>
  </body>
</html>
```

</TabItem>
</Tabs>

#### Penjelasan

Contoh di atas akan memilih semua elemen yang berada di dalam elemen dengan class `.main1`.

Jika kita ingin memmilh elemen tertentu bisa tentukan selectornya contoh hanya memilih semua elemen dengan tag `<p>` maka `.main p:nth-last-child(n)`.

## 2. :nth-last-child(A)

Berfungsi untuk memilih elemen child berdasarkan index/urutan tertentu dari parentnya, dengan cara mengisi posisi index elemen yang ingin di pilih.

- argumen (A) untuk menampung (bilangan bulat) yang berfungsi untuk memilih elemen child yang tepat/spesifik.

<Tabs>
<TabItem value="DEMO" label="DEMO" default>

<Codepen dataSlugHash="MWRWdGM" title="pseudo class - nth-last-child(A)" />

</TabItem>

<TabItem value="COPY CODE" label="COPY CODE">

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>:nth-last-child(A)</title>
    <style>
      li:nth-last-child(2) {
        color: green;
      }
    </style>
  </head>
  <body>
    <ul>
      <li>elemen li 1</li>
      <li>elemen li 2</li>
      <li>elemen li 3</li>
      <li>elemen li 4</li>
      <li>elemen li 5</li>
    </ul>
  </body>
</html>
```

</TabItem>
</Tabs>

#### Penjelasan

Contoh di hanya memilih elemen `li ke 4` karena elemen `li ke 4` merupakan child/anak ke 2 dari parent-nya yaitu `ul` jika di hitung dari belakang/urutas paling terakhir.

## 3. :nth-last-child(An)

Berfungsi untuk memilih elemen child berdasarkan angka kelipatan.

- argumen (A) untuk menampung (bilangan bulat) yang befungsi untuk memilih elemen dari setiap kelipatan - nya.
- argumen (n) untuk menampung bilangan bulat/urutan elemen yang nilainya dimulai dari (0), jadi n ke 0 untuk elemen 1, n ke 1 untuk elemen 2 dan seterusnya hingga semua elemen habis.

```txt
- (An) A * n
- (3n) 3 * n = ?
```

<Tabs>
<TabItem value="DEMO" label="DEMO" default>

<Codepen dataSlugHash="gOyOJKv" title="pseudo class - nth-last-child(An)" />

</TabItem>

<TabItem value="COPY CODE" label="COPY CODE">

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>:nth-last-child(An)</title>
    <style>
      li:nth-last-child(3n) {
        color: red;
        text-decoration: underline;
      }
    </style>
  </head>
  <body>
    <ul>
      <li>elemen li 1</li>
      <li>elemen li 2</li>
      <li>elemen li 3</li>
      <li>elemen li 4</li>
      <li>elemen li 5</li>
      <li>elemen li 6</li>
      <li>elemen li 7</li>
      <li>elemen li 8</li>
      <li>elemen li 9</li>
      <li>elemen li 10</li>
    </ul>
  </body>
</html>
```

</TabItem>
</Tabs>

#### Penjelasan

- n ke 0 : `3 * 0 = 0` tidak dipilih karena urutan elemen ke 0 tidak ada.
- n ke 1 : `3 * 1 = 3` di pilih karena urutan elemen ke 3 ada
- n ke 2 : `3 * 2 = 6` di pilih karena urutan elemen ke 6 ada
- n ke 3 : `3 * 3 = 9` di pilih karena urutan elemen ke 9 ada

Maka dari itu urutan elemen yang akan di pilih adalah 8,5 dan 2 karena perhitungan urutan elemen di mulai dari elemen paling terakhir itulah kenapa bukan elemen 3,6 dan 9 yang di pilih.

## 4. :nth-last-child(An+B)

Berfungsi untuk memilih elemen child berdasarkan angka kelipatan.

- argumen (A) untuk menampung (bilangan bulat) yang befungsi untuk memilih elemen dari setiap kelipatan - nya.
- argumen (n) untuk menampung bilangan bulat/urutan elemen yang nilainya dimulai dari (0), jadi n ke 0 untuk elemen 1, n ke 1 untuk elemen 2 dan seterusnya hingga semua elemen habis.
- (+) untuk menjumlahkan hasil dari (A*n) dengan (B).
- argumen (B) untuk menampung (bilangan bulat) yang berfungsi sebagai offset/pergeseran urutan elemen atau menentukan pilihan awal pada elemen.

```txt
- (An+B) A * n + B
- (2n+1) 2 * n + 1 = ?
```

</TabItem>

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>:nth-last-child(An+B)</title>
    <style>
      ul :nth-last-child(2n + 1) {
        color: blue;
      }
    </style>
  </head>
  <body>
    <ul>
      <li>li 1</li>
      <li>li 2</li>
      <li>li 3</li>
      <li>li 4</li>
      <li>li 5</li>
      <li>li 6</li>
      <li>li 7</li>
      <li>li 8</li>
      <li>li 9</li>
      <li>li 10</li>
    </ul>
  </body>
</html>
```

</TabItem>
</Tabs>

#### Penjelasan

- n ke 0 : `(2 * 0) + 1 = 1` di pilih karena urutan elemen ke 1 ada
- n ke 1 : `(2 * 1) + 1 = 3` di pilih karena urutan elemen ke 3 ada
- n ke 2 : `(2 * 2) + 1 = 5` di pilih karena urutan elemen ke 5 ada
- n ke 3 : `(2 * 3) + 1 = 7` di pilih karena urutan elemen ke 7 ada
- n ke 4 : `(2 * 4) + 1 = 9` di pilih karena urutan elemen ke 9 ada

Maka dari itu urutan elemen yang akan di pilih adalah 10,8,6,4 dan 2 karena perhitungan urutan elemen di mulai dari elemen paling terakhir itulah kenapa bukan elemen 1,3,5,7 dan 9 yang di pilih.

## 5. :nth-last-child(An-B)

Berfungsi untuk memilih elemen child berdasarkan angka kelipatan, pseudo ini kebalikan dari (An+B).

- argumen (A) untuk menampung (bilangan bulat) yang befungsi untuk memilih elemen dari setiap kelipatan - nya.
- argumen (n) untuk menampung bilangan bulat/urutan elemen yang nilainya dimulai dari (0), jadi n ke 0 untuk elemen 1, n ke 1 untuk elemen 2 dan seterusnya hingga semua elemen habis.
- (-) untuk mengurangi hasil dari (A*n) dengan (B).
- argumen (B) untuk menampung (bilangan bulat) yang berfungsi sebagai offset/pergeseran urutan elemen atau menentukan pilihan awal pada elemen.

```txt
- (An-B) A * n - B
- (2n-1) 2 * n - 1 = ?
```

</TabItem>

```html

  <head>
    <title>:nth-last-child(An-B)</title>
    <style>
      li:nth-last-child(3n - 1) {
        color: blue;
      }
    </style>
  </head>
  <body>
    <ul>
      <li>li 1</li>
      <li>li 2</li>
      <li>li 3</li>
      <li>li 4</li>
      <li>li 5</li>
      <li>li 6</li>
      <li>li 7</li>
      <li>li 8</li>
      <li>li 9</li>
      <li>li 10</li>
    </ul>
  </body>
</html>
```

</TabItem>
</Tabs>

#### Penjelasan

- n ke 0 : `(3 * 0) - 1 = -1` tidak di pilih karena elemen ke -1 tidak ada
- n ke 1 : `(3 * 1) - 1 = 2` di pilih karena elemen ke 2 ada
- n ke 2 : `(3 * 2) - 1 = 5` di pilih karena elemen ke 5 ada
- n ke 3 : `(3 * 3) - 1 = 8` di pilih karena elemen ke 8 ada

Maka dari itu elemen yang akan di pilih adalah 9,6 dan 3 karena perhitungan urutan elemen di mulai dari urutan elemen paling terakhir itulah kenapa bukan elemen 2,5 dan 8 yang di pilih.

## 8. :nth-last-child(-n+B)

Berfungsi untuk memilih beberapa elemen child sekaligus dengan berurutan dan tanpa kelipatan.

<Tabs>
<TabItem value="DEMO" label="DEMO" default>

<Codepen dataSlugHash="XWQWwyb" title="pseudo class - nth-last-child(-n+B)" />

</TabItem>

<TabItem value="COPY CODE" label="COPY CODE">

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>:nth-last-child(-n+B)</title>
    <style>
      li:nth-last-child(-n + 5) {
        color: red;
      }
    </style>
  </head>
  <body>
    <ul>
      <li>li 1</li>
      <li>li 2</li>
      <li>li 3</li>
      <li>li 4</li>
      <li>li 5</li>
      <li>li 6</li>
      <li>li 7</li>
      <li>li 8</li>
      <li>li 9</li>
      <li>li 10</li>
    </ul>
  </body>
</html>
```

</TabItem>
</Tabs>

#### Penjelasan

- n ke 0 = (-0) + 5 = 5 akan dipilih karena elemen ke 5 ada
- n ke 1 = (-1) + 5 = 4 akan dipilih karena elemen ke 4 ada
- n ke 2 = (-2) + 5 = 3 akan dipilih karena elemen ke 3 ada
- n ke 3 = (-3) + 5 = 2 akan dipilih karena elemen ke 2 ada
- n ke 4 = (-4) + 5 = 1 akan dipilih karena elemen ke 1 ada
- n ke 5 = (-5) + 5 = 0 tidak akan di pilih karena elemen ke 0 tidak ada

Jika variabel (B) sudah bernilai 0 maka pengecekan setiap urutan elemen akan di hentikan dan karena perhitungan di mulai dari urutan elemen child terakhir maka elemen child yang akan dipilih adalah 10,9,8,7 dan 6.
