<hr />

**`:required`** adalah sebuah pseudo-class yang berfungsi untuk memberi style pada elemen input yang bersifat required.

:::info CATATAN

- pseudo ini hanya berfungsi pada elemen input.

- wajib menambahkan atribut required.

:::

#### Contoh

<Tabs>
<TabItem value="DEMO" label="DEMO" default>

<Codepen dataSlugHash="oNOxGby" title="pseudo class - required" />

</TabItem>

<TabItem value="COPY CODE" label="COPY CODE">

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>:required</title>
    <style>
      input:required {
        color: rgb(255, 8, 0);
        border-color: red;  
      }
    </style>
  </head>
  <body>
    <div>
      <label for="address1">Alamat 1 : </label>
      <input type="text" id="address1" placeholder="required" required/>
    </div>
    <br /><br />
    <div>
      <label for="address2">Alamat 2 : </label>
      <input type="text" id="address2" placeholder="optional" />
    </div>
  </body>
</html>
```

</TabItem>
</Tabs>

#### Penjelasan

Contoh di atas akan menerapkan style pada elemen `<input>` ke 1 karena sifatnya required (wajib di isi), sementara elemen `<input>` ke 2 bersifat optional (tidak wajib di isi).
