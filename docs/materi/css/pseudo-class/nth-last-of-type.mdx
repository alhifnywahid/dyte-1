<hr />

**`:nth-last-of-type()`** fungsi-nya sama dengan pseudo :nth-of-type(), perbedaan-nya hanya pada pemulaian pada urutan elemen dimana :nth-last-of-type() ini akan di mulai dari elemen child terakhir dari induk atau parent-nya.

- 1. :nth-last-of-type(n)
- 2. :nth-last-of-type(A)
- 3. :nth-last-of-type(An)
- 4. :nth-last-of-type(An+B)
- 5. :nth-last-of-type(An-B)
- 6. :nth-last-of-type(odd)
- 7. :nth-last-of-type(even)
- 8. :nth-last-of-type(-n+B)

Untuk penjelasan argumen-nya silahkan cek di [`:nth-of-type(arg)`](./nth-of-type.mdx)

:::info CATATAN

- pseudo ini berfungsi pada semua elemen.

:::

## 1. :nth-last-of-type(n)

berfungsi untuk memilih beberapa elemen tertentu ataupun semua elemen yang berada di dalam parentnya.

<Tabs>
<TabItem value="DEMO" label="DEMO" default>

<Codepen dataSlugHash="XWQdbba" title="pseudo class - nth-last-of-type(n)" />

</TabItem>

<TabItem value="COPY CODE" label="COPY CODE">

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>:nth-last-of-type(n)</title>
    <style>
      div :nth-last-of-type(n){
        color: red;
      }
    </style>
  </head>
  <body>
    <div>
      <p>paragraf 1</p>
      <span>span 1</span>
      <p>paragraf 2</p>
      <span>span 2</span>
      <p>paragraf 3</p>
      <span>span 3</span>
    </div>
  </body>
</html>
```

</TabItem>
</Tabs>

#### Penjelasan

Contoh di atas akan memilih semua elemen child dari elemen `<div>`.

Jika kita ingin memmilh elemen tertentu bisa tentukan selectornya contoh hanya memilih semua elemen dengan tag `<p>` maka `div p:nth-of-type(n)`.

## 2. :nth-last-of-type(A)

Berfungsi untuk memilih elemen child berdasarkan index/urutan tertentu dari parentnya, dengan cara mengisi posisi index elemen yang ingin di pilih.

- argumen (A) untuk menampung (bilangan bulat) yang berfungsi untuk memilih elemen child yang tepat/spesifik.

<Tabs>
<TabItem value="DEMO" label="DEMO" default>

<Codepen dataSlugHash="NWmNqyr" title="pseudo class - nth-last-of-type(A)" />

</TabItem>

<TabItem value="COPY CODE" label="COPY CODE">

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>:nth-last-of-type(A)</title>
    <style>
      div :nth-last-of-type(2) {
        color: red;
      }
    </style>
  </head>
  <body>
    <div>
      <p>paragraf 1</p>
      <span>paragraf 1</span>
      <p>paragraf 2</p>
      <span>paragraf 2</span>
      <p>paragraf 3</p>
      <span>paragraf 3</span>
      <p>paragraf 4</p>
      <span>paragraf 4</span>
    </div>
  </body>
</html>
```

</TabItem>
</Tabs>

#### Penjelasan

Contoh di atas hanya memilih elemen `<paragraf>` dan `<span>` ke 3 karena posisi nya berada di urutan ke 2 jika di hitung dari index urutan elemen paling terakhir.

## 3. :nth-last-of-type(An)

Berfungsi untuk memilih elemen child berdasarkan angka kelipatan.

- argumen (A) untuk menampung (bilangan bulat) yang befungsi untuk memilih elemen dari setiap kelipatan - nya.
- argumen (n) untuk menampung bilangan bulat/urutan elemen yang nilainya dimulai dari (0), jadi n ke 0 untuk elemen 1, n ke 1 untuk elemen 2 dan seterusnya hingga semua elemen habis.

```txt
- (An) A * n
- (2n) 2 * n = ?
```

<Tabs>
<TabItem value="DEMO" label="DEMO" default>

<Codepen dataSlugHash="yLrONvX" title="pseudo class - nth-last-of-type(An)" />

</TabItem>

<TabItem value="COPY CODE" label="COPY CODE">

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>:nth-last-of-type(An)</title>
    <style>
      div :nth-last-of-type(2n){
        color: red;
        text-decoration: underline;
      }
    </style>
  </head>
  <body>
    <div>
      <span>span 1</span>
      <p>paragraf 1</p>
      <span>span 2</span>
      <p>paragraf 2</p>
      <span>span 3</span>
      <p>paragraf 3</p>
      <span>span 4</span>
      <p>paragraf 4</p>
      <span>span 5</span>
      <p>paragraf 5</p>
      <span>span 6</span>
      <p>paragraf 6</p>
    </div>
  </body>
</html>
```

</TabItem>
</Tabs>

#### Penjelasan

- n ke 0 : `3 * 0 = 0` tidak dipilih karena urutan elemen ke 0 tidak ada.
- n ke 1 : `3 * 1 = 3` di pilih karena urutan elemen ke 3 ada
- n ke 2 : `3 * 2 = 6` di pilih karena urutan elemen ke 6 ada
- n ke 3 : `3 * 3 = 9` di pilih karena urutan elemen ke 9 ada

Maka dari itu urutan elemen yang akan di pilih adalah `<span>` dan `<paragraf>` 5,3 dan 1 karena perhitungan urutan elemen di mulai dari elemen paling terakhir itulah kenapa bukan elemen 2,4 dan 6 yang di pilih.

## 4. :nth-last-of-type(An+B)

Berfungsi untuk memilih elemen child berdasarkan angka kelipatan.

- argumen (A) untuk menampung (bilangan bulat) yang befungsi untuk memilih elemen dari setiap kelipatan - nya.
- argumen (n) untuk menampung bilangan bulat/urutan elemen yang nilainya dimulai dari (0), jadi n ke 0 untuk elemen 1, n ke 1 untuk elemen 2 dan seterusnya hingga semua elemen habis.
- (+) untuk menjumlahkan hasil dari (A*n) dengan (B).
- argumen (B) untuk menampung (bilangan bulat) yang berfungsi sebagai offset/pergeseran urutan elemen atau menentukan pilihan awal pada elemen.

```txt
- (An+B) A * n + B
- (2n+1) 2 * n + 1 = ?
```

```html
    </style>
```

</TabItem>

#### Penjelasan

- n ke 0 : `(2 * 0) + 1 = 1` di pilih karena urutan elemen ke 1 ada
- n ke 1 : `(2 * 1) + 1 = 3` di pilih karena urutan elemen ke 3 ada
- n ke 2 : `(2 * 2) + 1 = 5` di pilih karena urutan elemen ke 5 ada

Maka dari itu urutan elemen yang akan di pilih adalah elemen `<paragraf>` dan `<span>` ke  6,4 dan 2 karena perhitungan urutan elemen di mulai dari elemen paling terakhir itulah kenapa bukan elemen 1,3 dan 5 yang di pilih.

## 5. :nth-last-of-type(An-B)

Berfungsi untuk memilih elemen child berdasarkan angka kelipatan, pseudo ini kebalikan dari (An+B).

- argumen (A) untuk menampung (bilangan bulat) yang befungsi untuk memilih elemen dari setiap kelipatan - nya.
- argumen (n) untuk menampung bilangan bulat/urutan elemen yang nilainya dimulai dari (0), jadi n ke 0 untuk elemen 1, n ke 1 untuk elemen 2 dan seterusnya hingga semua elemen habis.
- (-) untuk mengurangi hasil dari (A*n) dengan (B).
- argumen (B) untuk menampung (bilangan bulat) yang berfungsi sebagai offset/pergeseran urutan elemen atau menentukan pilihan awal pada elemen.

```txt
- (An-B) A * n - B
- (2n-1) 2 * n - 1 = ?
```

#### Penjelasan

- n ke 2 : `(3 * 2) - 1 = 5` di pilih karena elemen ke 5 ada
- n ke 3 : `(3 * 3) - 1 = 8` di pilih karena elemen ke 8 ada

Maka dari itu elemen yang akan di pilih adalah elemen `<paragraf>` dan `<span>` ke 7,4 dan 1 karena perhitungan urutan elemen di mulai dari urutan elemen paling terakhir itulah kenapa bukan elemen 2,5 dan 8 yang di pilih.

## 8. :nth-last-of-type(-n+B)

Berfungsi untuk memilih beberapa elemen child sekaligus dengan berurutan dan tanpa kelipatan.

<Tabs>
<TabItem value="DEMO" label="DEMO" default>

<Codepen dataSlugHash="poByJVw" title="pseudo class - nth-last-of-type(-n+B)" />

</TabItem>

<TabItem value="COPY CODE" label="COPY CODE">

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>:nth-last-of-type(-n+B)</title>
    <style>
      div :nth-last-of-type(-n+3){
        color: red;
        text-decoration: underline;
      }
    </style>
  </head>
  <body>
    <div>
      <span>span 1</span>
      <p>paragraf 1</p>
      <span>span 2</span>
      <p>paragraf 2</p>
      <span>span 3</span>
      <p>paragraf 3</p>
      <span>span 4</span>
      <p>paragraf 4</p>
      <span>span 5</span>
      <p>paragraf 5</p>
      <span>span 6</span>
      <p>paragraf 6</p>
    </div>
  </body>
</html>
```

</TabItem>
</Tabs>

#### Penjelasan

- n ke 0 = (-0) + 3 = 3 akan dipilih karena elemen ke 3 ada
- n ke 1 = (-1) + 3 = 2 akan dipilih karena elemen ke 2 ada
- n ke 2 = (-2) + 3 = 1 akan dipilih karena elemen ke 1 ada
- n ke 3 = (-3) + 3 = 0 tidak akan di pilih karena elemen ke 0 tidak ada

Jika variabel (B) sudah bernilai 0 maka pengecekan setiap urutan elemen akan di hentikan dan karena perhitungan di mulai dari urutan elemen child terakhir maka elemen child yang akan dipilih elemen `<paragraf>` dan `<span>` ke 6,5 dan 4.
