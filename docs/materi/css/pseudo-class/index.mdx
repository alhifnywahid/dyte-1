# CSS Pesudo-Class

Pseudo-class adalah kata kunci yang menambahkan gaya ke elemen berdasarkan keadaan spesifik elemen tersebut.

### Cara Penulisan Code 

```css 
selector:pseudo-class{
  /* style yang ingin di berikan */
}
```
