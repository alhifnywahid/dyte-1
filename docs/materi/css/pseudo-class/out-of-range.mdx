<hr />

**`:out-of-range`** adalah sebuah pseudo-class yang berfungsi untuk memberi style pada elemen input yang brtipe number & memiliki atribut min & max, dimana elemen tersebut akan berubah ketika user memasukkan angka ketika di luar jangkauan min & max yang telah di tentukan.

:::info CATATAN

- pseudo ini hanya berfungsi pada elemen input yang bertipe number dengan syarat mempunyai atribut min & max

- agar style tidak di terapkan terlebih dahulu kita bisa menambahkan atribut `required` pada elemen `<input>` tsb.

:::

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>:out-of-range</title>
    <style>
      input {
        border: 2px solid rgb(21, 252, 0);
      }
      input:out-of-range {
        border-color: rgb(255, 0, 0);
      }
    </style>
  </head>
  <body>
    <label for="number">Masukkan Angka : </label>
    <input
      type="number"
      name="number"
      id="number"
      min="6"
      max="10"
      value="0"
    />
  </body>
</html>
```

#### Penjelasan

Contoh di atas akan menerapkan style pada elemen `<input>` jika bilangan bulat yang di masukkan adalah 6 - 10 maka warna bordernya akan berubah menjadi hijau, jika di luar 6 - 10 maka warna bordernya akan menjadi merah.
