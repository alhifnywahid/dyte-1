## Elemen/Tag Selector

Berfungsi memilih elemen html berdasarkan tagnya.

### Contoh

- Tuliskan nama tag html yang ingin di ubah style nya.

<Codepen dataSlugHash="xxBeoRo" title="simple selector - tag" />

### Penjelasan

Contoh di atas akan memilih semua elemen html yang menggunakan tag `h1` saja dan mengubah warnanya menjadi hijau, elemen selain `h1` warnanya akan tetap hitam.

## Class Selector

Berfungsi untuk memilih elemen html berdasarkan class.

### Contoh

- Tuliskan symbol titik (.) lalu tuliskan nama class-nya.

<Codepen dataSlugHash="eYXowex" title="simple selector - class" />

### Penjelasan

Contoh di atas akan memilih elemen `list ke 1 - 3` karena mempunyai class yang cocok.

## ID Selector

Berfungsi untuk memilih elemen html berdasarkan id.

### Contoh

- Tuliskan symbol pagar (#) lalu tuliskan nama id-nya.

<Codepen dataSlugHash="WNmWqyy" title="simple selector - id" />

### Penjelasan

Contoh di atas akan memilih `list ke 3 - 5` karena mempunyai id yang cocok.

## Universal Selector

Berfungsi untuk memilih semua elemen yang ada di html (tanpa terkecuali) baik mempunyai class/id maupun tidak mempunyai class/id

### Contoh

- Tuliskan symbol bintang (*) untuk memilih semua elmen yang ada.

<Codepen dataSlugHash="BabEgqY" title="simple selector - universal" />


### Penjelasan

Contoh di atas akan memilih semua elemen dan mengubah warna teksnya menjadi biru.
