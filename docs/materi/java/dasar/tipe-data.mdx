tipe data berfungsi untuk menentukan data pada sebuah variabel contoh bilangan bulat menggunakan tipe data `int`, bilangan pecahan menggunakan tipe data `float`.

di java sendiri ada banyak macam tipe data di antaranya sebagai berikut ini :

![f](/img/materi/tipedata-java.png)
