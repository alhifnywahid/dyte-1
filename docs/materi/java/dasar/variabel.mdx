Variabel berfungsi untuk menyimpan sebuah data atau nilai yang beresifat sementara.

## Aturan penulisan variabel

### Larangan
- tidak boleh di awali dengan angka.
- tidak boleh angka semua.
- tidak boleh ada spasi.
- tidak boleh menggunakan angka yang telah di pakai oleh java seperti `args`, `static`, `public` dan masih banyak lagi.

### Disarankan
- tulis variabel dengan huruf kecil.
- gunakan camel cases dimana ketika nama variabel berupa kalimat atau lebih dari 1 kata maka huruf pertama di kata pertama berhuruf kecil, dan huruf pertama di kata selanjutnya menggunakan huruf besar.

  contoh : 
  - Nama Lengkap : namaLengkap
  - Pintu Bagian Belakang : pintuBagianBelakang


- boleh menggunakan simbol di bagian mana saja.
- boleh menggunakan angka dimana saja kecuali di awal nama variabel.

### contoh
:::tip penulisan variabel yang benar

```java
// menggunakan abjad semua
String nama = "Alhifny"; 

// jika lebih dari 1 kata gunakan teknik camel cases
String namaLengkap = "Alhifny Wahid"; 

// boleh menggunakan angka di belakang
String nama1 = "Alhifny"; 

// boleh menggunakan angka di belakang
String nama2 = "Wahid"; 
```
:::

<hr />

:::danger penulisan variabel yang salah

```java
// tidak boleh menggunakan angka semua
String 12345 = "Alhifny Wahid"; 

// tidak boleh di awali angka
String 1nama = "Alhifny Wahid"; 

// tidak boleh menggunakan spasi
String nama lengkap = "Alhifny Wahid"; 

// tidak boleh menggunakan kata yang telah di pakai oleh java.
String args = "Alhifny Wahid"; 
```
:::
