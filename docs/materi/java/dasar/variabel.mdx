# Variabel

### fungsi variabel
Variabel berfungsi untuk menyimpan sebuah data atau nilai yang beresifat sementara.

### penulisan variabel
- buat tipe data dulu lalu buat nama variabel dan isi nilai dari variabel sesuai tipe data.
- kita bebas memberi nama apa saja pada variabel keculai menggunakan nama yang telah di pakai oleh java seperti `args`, `main`, dan masih banyak lagi.
- tidak boleh di awali angka atau berisi angka semua.
- tidak boleh menggunakan spasi, disarankan jika nama variabel melebihi dari 1 kata maka huruf pertama di kata pertama menggunakan huruf kecil dan huruf kedua di kata ke dua menggunakan huruf besar.

### contoh
:::tip penulisan variabel yang benar

```java
// menggunakan abjad semua
String nama = "Alhifny"; 

// jika lebih dari 1 kata gunakan teknik camel cases
String namaLengkap = "Alhifny Wahid"; 

// boleh menggunakan angka di belakang
String nama1 = "Alhifny"; 

// boleh menggunakan angka di belakang
String nama2 = "Wahid"; 
```
:::

<hr />

:::danger penulisan variabel yang salah

```java
// tidak boleh menggunakan angka semua
String 12345 = "Alhifny Wahid"; 

// tidak boleh di awali angka
String 1nama = "Alhifny Wahid"; 

// tidak boleh menggunakan spasi
String nama lengkap = "Alhifny Wahid"; 

// tidak boleh menggunakan kata yang telah di pakai oleh java.
String args = "Alhifny Wahid"; 
```
:::
