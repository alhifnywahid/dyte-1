Variabel berfungsi untuk menyimpan sebuah data atau nilai yang beresifat sementara.

Java adalah bahasa static type, sehingga sebuah  variabel hanya bisa digunakan untuk menyimpan tipe data yang sama, tidak bisa berubah-ubah tipe data seperti di bahasa pemrograman PHP atau JavaScript

## Aturan penulisan variabel


Untuk membuat variabel di Java kita bisa menuliskan nama tipe datanya lalu nama variabelnya.

Setiap bahasa pemerograman biasanya mempunyai aturan - aturan dalam penulisan sintaks variabel.

Berikut aturan penulisan variabel yang di sarankan / di perbolehkan / yang dilarang.

<Tabs>
<TabItem value="disarankan" label="Di Sarankan" default>

:::tip contoh penulisan variabel yang di sarankan.

Penulisan variabel pada java sangat di sarankan sebagai berikut ini : 

- menggunakan huruf kecil semua.

  contoh : 
  ```java
  String nama = "Hifny";
  ```
- gunakan camel case jika lebih dari 1 kata. (misal ada 2 kata, maka huruf pertama di kata ke dua gunakan huruf kapital)

  contoh : 
  ```java
  String namaLengkap = "Alhifny Wahid";
  ```

:::

</TabItem>
<TabItem value="diperbolehkan" label="Di Perbolehkan">

:::info contoh penulisan variabel yang di perbolehkan.

Penulisan variabel pada java sangat di sarankan sebagai berikut ini : 

- boleh menggunakan huruf kecil, besar, atau campuran sekaligus.

  contoh : 
  ```java
  String NAMADEPAN = "Ahmad";
  String namabelakang = "Iham";
  String NamaLengkap = "Ahmad Ilham";
  ```
- boleh menggunakan campuran simbol di awal, di tengah baik di akhir kata. (hanya boleh menggunakan beberapa simbol seperti `_ $`)

  contoh : 
  ```java
  String $NamaDepan = "Ahmad";
  String Nama_Belakang = "Ilham";
  String _namaLengkap$ = "Ahmad Ilham";
  ```
- boleh menggunakan campuran angka di setelah huruf pertama.

  contoh : 
  ```java
  String nama1 = "Ahmad";
  String nama2nya = "Ilham";
  String namalenkap3 = "Ahmad Ilham";
  ```

:::

</TabItem>
<TabItem value="dilarang" label="Di Larang">

:::danger contoh penulisan variabel yang di di larang.

Pernulisan variabel pada java sangat di sarankan sebagai berikut ini : 

- tidak boleh di meggunakan kata yang telah di pakai oleh java seperti (args, static final dll).

  contoh : 
  ```java
  String args = "Ilham Saifuddin";
  String static = "Fernando Imam";
  ```

- tidak boleh di awali angka.

  contoh : 
  ```java
  String 1nama = "Ilham Saifuddin";
  String 2nama = "Fernando Imam";
  ```
- tidak boleh menggunakan angka semua.

  contoh : 
  ```java
  String 12345 = "Ahmad";
  ```
- tidak boleh menggunakan spasi, enter dan tab.

  contoh : 
  ```java
  String nama depan = "Ilham";
  String nama belakang = "Saifuddin";
  String nama lengkap = "Ilham Saifuddin";
  ```

:::
