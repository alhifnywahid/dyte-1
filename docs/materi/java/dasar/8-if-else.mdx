Dalam bahasa pemrograman Java, `if statement` digunakan untuk menjalankan blok kode tertentu jika suatu kondisi bernilai true. Jika kondisi bernilai false, blok kode tersebut akan dilewati.

Dengan percabangan inilah kita bisa membuat program dengan beberapa alur.

Kalau kita perhatikan, alur pengeksekusian sebuah kode program dikerjakan satu per satu dari atas sampai ke bawah.

Baris demi baris dibaca, kemudian komputer mengerjakan apa yang diperintahkan(kode yang telah di tuliskan).

Misalnya seperti :

![percabangan](/img/java/alur-program-tanpa-percabangan.png)

Alur programnya satu, tidak ada belokan atau percabangan.

Pada diagram alur (Flow Chart) pada gambar di atas, alurnya memang satu.

Tapi setelah kita menggunakan percabangan, alurnya akan bertambah menjadi seperti berikut :

![percabangan](/img/java/alur-program-dengan-percabangan.png)

`if-statement` mempunyai `3` jenis sintaks yang bisa kita tuliskan untuk mengatur alur program yaitu :

- `if`
- `else if`
- `else`

Berikut sintaks penulisan percabangan `if statement` dalam bahasa Java:

<Tabs>
<TabItem value="if statement" default>

**if statement**

Sintaks `if` ini berfungsi untuk membuat 2 percabangan saja.

Sintaks `if` membutuhkan sebuah kondisi, dimana jika kondisi `true` program akan mengeksekusi perintah yang ada di dalam kode blok yang telah kita tuliskan.

Sintaks `if` hanya bisa di tuliskan 1 kali saja.

#### Penulisan Sintaks

```java
if (kondisi){
  // Kode yang akan dieksekusi jika kondisi benar
}
```

#### Contoh Program

Contoh program sederhana alur `lampu lalu lintas`.

```java showLineNumbers title="Main.java"
public class Main {
  public static void main(String[] args){
    String lampu = "merah";
    if (lampu == "merah"){
      System.out.println("Lampu merah, Silahkan Berhenti.");
    }
  }
}
```

#### Output Program

```bash tittle="Output"
Lampu merah, Silahkan Berhenti.
```

#### Penjelasan Program

Dari contoh di atas seharusnya kita sudah tau apa yang akan di tampilkan ke layar oleh program.

- `Baris ke 3` : Inisialisasi variabel lampu = "merah"
- `Baris ke 4` : Mengecek apakah variabel lampu = merah ?
  - `Jika benar` (variabel lampu = merah) : akan menampilkan pesan `"Lampu merah, Silahkan Berhenti."` ke layar
  - `Jika salah` (variabel lampu bernilai selain merah) : tidak akan menampilkan apapun ke layar karena tidak ada kode yang di eksekusi jika lampu bernilai selain merah.

**else if steatment**

Sintaks `else if` ini berfungsi untuk membuat lebih dari 2 percabangan.

Sintaks `else if` juga membutuhkan sebuah kondisi, dimana jika kondisi `true` program akan mengeksekusi perintah yang ada di dalam kode blok yang telah kita tuliskan.

Sintaks `else if` bisa kita tuliskan lebih dari 1 kali.

Sintask `else if` tidak bisa di tuliskan tanpa sintaks `if`

#### Penulisan Sintaks

```java
if (kondisi pertama){
  // Kode yang akan dieksekusi jika kondisi pertama benar
} else if (konsisi kedua) {
  // Kode yang akan dieksekusi jika kondisi pertama salah dan kondisi kedua benar
}
```

#### Contoh Program

Contoh program sederhana alur `lampu lalu lintas`. menambah lampu 'kuning'.

```java showLineNumbers title="Main.java"
public class Main {
  public static void main(String[] args){
    String lampu = "kuning";
    if (lampu == "merah"){
      System.out.println("Lampu merah, Silahkan Berhenti.");
    } else if (lampu == "kuning"){
      System.out.println("Lampu kuning, Hati Hati.");
    }
  }
}
```

#### Output Program

```bash tittle="Output"
Lampu kuning, Hati Hati.
```

#### Penjelasan Program

- `Baris ke 3` : Inisialisasi variabel lampu = "kuning"
- `Baris ke 4` : Mengecek apakah variabel lampu = merah ?
  - `Jika benar` (variabel lampu = merah) : akan menampilkan pesan `"Lampu merah, Silahkan Berhenti."` ke layar
  - `Jika salah` (variabel lampu bernilai selain merah) : program akan melanjutkan pengecekan ke kondisi ke dua yaitu di sintaks `else if` pada baris ke 6.
- `Baris ke 6` : Mengecek apakah variabel lampu = kuning ?
  - `Jika benar` (variabel lampu = kuning) : akan menampilkan pesan `"Lampu kuning, Hati Hati."` ke layar
  - `Jika salah` (variabel lampu bernilai selain kuning) : tidak akan menampilkan apapun ke layar karena semua kondisi di baris ke `3` dan `6` salah otomatis tidak akan ada kode yang akan di eksekusi.

**else steatment**

Sintaks `else` ini berfungsi untuk mengeksekusi program jika semua kondisi `if` atau `esle if` `salah` atau `false`.

Sintaks `else` tidak membutuhkan sebuah kondisi, program akan otomatis mengeksekusi semua kode atau perintah yang berada di dalam blok `else`

Sintaks `else` hanya bisa di tuliskan 1 kali saja, yaitu di akhir semua kondisi boleh di setelah `if` ataupun `else if` intinya di akhir semua kondidisi.

#### Penulisan Sintaks

```java

// setelah if
if (kondisi){
  // Kode yang akan dieksekusi jika kondisi benar
} else {
  // Kode yang akan dieksekusi jika kondisi if salah
}

//setelah else if
if (kondisi pertama){
  // Kode yang akan dieksekusi jika kondisi benar
} else if (kondisi kedua) {
  // Kode yang akan dieksekusi jika kondisi pertama salah dan kondisi kedua ini benar
} else {
  // Kode yang akan dieksekusi jika kondisi pertama atau ke dua salah.
}
```

#### Contoh Program

Contoh program sederhana alur `lampu lalu lintas`.

```java showLineNumbers title="Main.java"
public class Main {
  public static void main(String[] args){
    String lampu = "hitam";
    if (lampu == "merah"){
      System.out.println("Lampu merah, Silahkan Berhenti.");
    } else if (lampu == "kuning"){
      System.out.println("Lampu kuning, Hati Hati.");
    } else if (lampu == "hijau"){
      System.out.println("Lampu hijau, Selamat Jalan.");
    } else {
      System.out.println("Maaf sepertinya lampu sedang rusak.");
    }
  }
}
```

#### Output Program

```bash tittle="Output"

```

#### Penjelasan Program

- `Baris ke 3` : Inisialisasi variabel lampu = "hitam"
- `Baris ke 4` : Mengecek apakah variabel lampu = merah ?
  - `Jika benar` (variabel lampu = merah) : akan menampilkan pesan `"Lampu merah, Silahkan Berhenti."` ke layar
  - `Jika salah` (variabel lampu bernilai selain merah) : program akan melanjutkan pengecekan ke kondisi ke dua yaitu di sintaks `else if` pada baris ke 6.
- `Baris ke 6` : Mengecek apakah variabel lampu = kuning ?
  - `Jika benar` (variabel lampu = kuning) : akan menampilkan pesan `"Lampu kuning, Hati Hati."` ke layar
  - `Jika salah` (variabel lampu bernilai selain kuning) : tidak akan menampilkan apapun ke layar karena semua kondisi di baris ke `3` dan `6` salah otomatis tidak akan ada kode yang akan di eksekusi.
- `Baris ke 8` : Mengecek apakah variabel lampu = hijau ?
  - `Jika benar` (variabel lampu = hijau) : akan menampilkan pesan `"Lampu hijau, Selamat Jalan."` ke layar
  - `Jika salah` (variabel lampu bernilai selain hijau) : otomatis akan menjalankan blok perintah di dalam blok `else` di baris ke `10`
