# Menampilkan data ke layar

Di java ada beberapa cara untuk menampilkan data seperti teks, angka atau nilai pada sebuah variabel.

Di sini saya hanya akan memberikan 1 cara yang sering di gunakan yaiut dengan method `System.out.println()`.

Method ini bisa menggunakan maksimal 1 parameter dan bisa juga tidak menggunakan parameter, dimana jika kita tidak menggunakan parameter program hanya akan mencetak 1 baris baru saja dan jika menggunakan parameter maka akan menampilkan data lalu akan menambahakan 1 baris baru.

**Apa itu parameter?**
```java
System.out.println(parameter)
```

Jadi tempat penulisan parameter itu di tengah tengah simbol `()` kata parameter itulah yang nanti bisa isi dengan teks, angka ataupun variabel.

<Tabs>
<TabItem value="Cetak Teks" default>

Kita bisa langsung mengisi parameter dengan teks yang kita inginkan dan teks tersbut harus di kutip dengan simbol (double quote) `""`.

**Contoh**

```java 
public class Main {
  public static void main(String[] args) {
    System.out.println("Bambang")
    System.out.println("Sugiono")
  }
}
``` 

**Output**

```bash
Bambang
Sugiono.
``` 

</TabItem>
<TabItem value="Cetak Angka">

Kita bisa langsung mengisi parameternya dengan angka tanpa di kutip simbol apapun.

**Contoh**

```java 
public class Main {
  public static void main(String[] args) {
    System.out.println(20)
  }
}
``` 

**Output**

```bash
20
``` 

</TabItem>
<TabItem value="VARIABEL">

Kita bisa langsung mengisi parameternya dengan nama variabel yang telah di buat.

**Contoh**

```java 
public class Main {
  public static void main(String[] args) {
    String nama = "Bambang Sugiono.";
    System.out.println(nama)
  }
}
``` 

**Output**

```bash
Bambang Sugiono.
``` 

</TabItem>
<TabItem value="GABUNGAN">

Kita bisa menggabungkan teks, angka variabel degan 1 method print saja

Cara menggabungkanny adalah dengan menambahkan simbol `+`

**Contoh**

```java 
public class Main {
  public static void main(String[] args) {
    String nama = "Bambang Sugiono";
    String alamat = "Surabaya.";

    System.out.println("Halo, Nama saya " + nama + " Saya berumur " + 20 + " Tahun " + "dan saya berasal dari " + alamat);
  }
}
``` 

**Output**

```bash
Halo, Nama saya Bambang Sugiono Saya berumur 20 Tahun dan saya berasal dari Surabaya.
``` 

</TabItem>
</Tabs>
