Operator adalah sebuah simbol atau karakter khusus yang di gunakan untuk melakukan operasi matematika, perbandingan, logika atau manipulasi data pada sebuah varaibel.

Misalkan kita ingin menjumlahkan nilai dari variabel `x` dan `y`, maka kita bisa menggunakan operator penjumlahan (+).

```bash
x + y
```

## Jenis Operator di Java

Di Java ada 6 jenis operator yaitu :

1. **Operator Artimatika.**
2. **Operator Penugasan.**
3. **Operator Pembanding.**
4. **Operator Logika.**
5. **Operator Bitwise.**
6. **Operator Ternary.**

#### Operator Aritmatika

Operator aritmatika digunakan untuk melakukan operasi aritmatika.

Operator ini terdiri dari:

| **Nama**    | **Simbol** |
| ----------- | ---------- |
| Penjumlahan | `+`        |
| Pengurangan | `-`        |
| Perkalian   | `*`        |
| Pembagian   | `/`        |
| Sisa Bagi   | `%`        |

##### Contoh Penggunaan

<Tabs>
<TabItem value="Penjumlahan">

Operator Penjumlahan di gunakan untuk menjumlahkan dua nilai atau ekspresi.

Operator Penjumlahan menggunakan simbol `+`

```java
public class Main {
  public static void main(String[] args) {

    // contoh 1
    byte angka1 = 20;
    byte angka2 = 20;
    byte hasil = (byte) (angka1 + angka2);

    System.out.println("Hasil Penjumlahan : " + hasil);

    // contoh 2
    System.out.println("Hasil Penjumlahan : " + (angka1 + angka2));

    // contoh 3
    System.out.println("Hasil Penjumlahan : " + (25 + 25));

    // contoh 4
    System.out.println("Hasil Penjumlahan : " + (angka1 + angka2 + hasil + 25));
  }
}
```

</TabItem>
<TabItem value="Pengurangan">

Operator Pengurangan di gunakan untuk mengurangkan dua nilai atau ekspresi.

Operator Pengurangan menggunakan simbol `-`

```java
    // contoh 1
    byte angka1 = 20;
    byte angka2 = 20;

    // contoh 2
    System.out.println("Hasil Pengurangan : " + (angka1 - angka2));

    // contoh 3
    System.out.println("Hasil Pengurangan : " + (25 - 25));

    // contoh 4
    System.out.println("Hasil Pengurangan : " + (angka1 - angka2 - hasil - 25));
  }
}
```

#### Operator Penugasan

Operator penugasan atau yang biasa di sebut (Assignment Operator) fungsinya untuk meberikan tugas pada variabel tertentu. Biasanya untuk mengisi nilai.

Operator Penugasan terdiri dari:

| **Nama**                  | **Simbol** |
| ------------------------- | ---------- |
| Pengisian Saja            | `=`        |
| Pengisian dan Penambahan  | `+=`       |
| Pengisian dan Pengurangan | `-=`       |
| Pengisian dan Perkalian   | `*=`       |
| Pengisian dan Pembagian   | `/=`       |
| Pengisian dan Sisa Bagi   | `%=`       |

##### Contoh Penggunaan

<Tabs>
<TabItem value="=">

Operator `=` di gunakan untuk mengisi nilai pada sebuah variabel.

```java
public class Main {
  public static void main(String[] args) {
    int a = 10;
    System.out.prinln("Isi Variabel a : " a) // Output 10

    a = 40;
    System.out.prinln("Isi Variabel a2 : " + a) // Output 40
  }
}
```

Awalnya variabel `a` di tugaskan untuk meyimpan nilai `10`. Dan nilai berubah menjadi 40 karena variabel `a` di tugaskan menyimpan nilai baru yaitu `40`.

</TabItem>
<TabItem value="+=">

Operator `+=` di gunakan untuk mengisi sekaligus menambah nilai pada sebuah variabel.

```java
public class Main {
  public static void main(String[] args) {
    int a = 10;
    System.out.prinln("Isi Variabel a : " a) // Output 10

    a += 40;
    System.out.prinln("Isi Variabel a2 : " + a) // Output 50
  }
}
```

Awalnya variabel `a` di tugaskan untuk meyimpan nilai `10`. Dan nilai berubah menjadi 50 karena variabel `a` di tugaskan menyimpan nilai `10` sekaligus menambahkan nilai baru yaitu `40`.

Operator ini sama halnya kita membuat :

```java
int a = 10;
a = a + 40;

// atau

int a = 10 + 40;
```


#### Operator Perbandingan

Operator perbandingan di gunakan untuk membandingkan 2 buah nilai atau lebih.

Operator perbandingan hanya memiliki 2 nilai, yaitu `true / benar` dan `false / salah`. Jadi kita bisa langsung menampilkan hasilnya atau bisa kita simpan di dalam variabel dengan tipe data `boolean`.

Operator ini terdiri dari:

| **Nama**                | **Simbol** |
| ----------------------- | ---------- |
| Sama Dengan             | `==`       |
| Lebih Besar             | `>`        |
| Lebih Besar Sama Dengan | `>=`       |
| Lebih Kecil             | `<`        |
| Lebih Kecil Sama Dengan | `<=`       |
| Tidak Sama Dengan       | `!=`       |

##### Contoh Penggunaan

<Tabs>
<TabItem value="==" default>

Operator `==` di gunakan untuk mengecek apakah 2 buah nilai sama atau tidak.

```java showLineNumbers
public class Main {
  public static void main(String[] args) {
    byte a = 10;

    System.out.println("Apakah 10 == 5 ? " + (a == 5)); // false

    System.out.println("Apakah 10 == 10 ? " + (a == 10)); // true

  }
}
```

- Di baris ke `5` bernilai `false` karena nilai pada variabel `a` tidak sama dengan 10.
- Di baris ke `7` bernilai `true` karena nilai pada variabel `a` sama dengan 10.

</TabItem>
<TabItem value=">">

- Di baris ke `5` bernilai `false` karena nilai pada variabel `a` tidak lebih besar dari `10`.

#### Operator Logika

Operator Logika di gunakan untuk membuat operasi logika seperti contoh :

- Pernyataan 1 : Ilham adalah seorang programer.
- Peryantaan 2 : Ilham menggunakan Linux.

Jika di tanya, Apakah ilham programer yang menggunakan linux?

Tentu kita akan cek dulu kebenarannya :

- Pernyataan 1 : Ilham adalah seorang programer = `true`
- Peryantaan 2 : Ilham menggunakan Linux = `true`

Apakah ilham programer yang menggunakan linux?

```bash
Peryataan 1 && Pernyataan 2 = true
```

Operator ini terdiri dari:

| **Nama**         | **Simbol** |
| ---------------- | ---------- | --- | --- |
| AND              | `&&`       |
| OR               | `          |     | `   |
| Negasi/Kebalikan | `!`        |

##### Contoh Penggunaan

<Tabs>
<TabItem value="AND" default>

Operator `AND` berfungsi untuk mengecek lebih dari 1 kondisi.

Operator `AND` akan bernilai `true` jika semua kondisi bernilai `true` dan sebaliknya akan bernilai `false` jika salah satu kondisi bernilai `false`.

| Pernyataan 1 | Pernyataan 2 | Logika AND |
| ------------ | ------------ | ---------- |
| true         | true         | true       |
| true         | false        | false      |
| false        | true         | false      |
| false        | false        | false      |

Misalnya seperti ini :

```bash
Pernyataan 1 : a = 10
Pernyataan 2 : b = 20
```

Jika ditanya, apakah variabel `a` = `10` dan variabel `b` = `20`?

Tentu kita akan cek dulu kebenarannya

```bash
Pernyataan 1 : a = 10 // true
Pernyataan 2 : b = 20 // true
```

Apakah variabel `a` = `10` dan variabel `b` = `20`?

```bash
Pertnyataan 1 && Pernyataan 2 = true
```

**Contoh Program**

```java showLineNumbers
public class Main {
  public static void main(String[] args){
    byte a = 10;
    byte b = 20;

    System.out.println("a = 10? dan b = 20? : " + (a == 10 && b == 20)); // true

    System.out.println("a = 10? dan b = 5? : " + (a == 10 && b == 5)); // false

    System.out.println("a = 7? dan b = 20? : " + (a == 7 && b == 20)); // false

    System.out.println("a = 7? dan b = 5? : " + (a == 10 && b == 20)); // false
  }
}
```

**Penjelasan Program**

- Baris ke `6` bernilai `true` karena semua pernyataan benar **a = 10 dan b =20**
- Baris ke `8` bernilai `false` karena pernyataan ke dua salah dimana variabel `b` tidak sama dengan `5`
- Baris ke `10` bernilai `false` karena pernyataan pertama salah dimana variabel `a` tidak sama dengan `7`
- Baris ke `12` bernilai `false` karena semua pernyataan salah dimana varaibel `a` tidak sama dengan `7` dan variabel `b` tidak sama dengan `5`

</TabItem>
<TabItem value="OR" default>

Operator `OR` berfungsi untuk mengecek lebih dari 1 pernyataan.

Operator `OR` akan bernilai `true` jika salah satu pernyataan bernilai `true` dan akan bernilai `false` jika semua pernyataan bernilai `false`.

| Pernyataan 1 | Pernyataan 2 | Logika OR |
| ------------ | ------------ | --------- |
| true         | true         | true      |
| true         | false        | true      |
| false        | true         | true      |
| false        | false        | false     |

Misalnya seperti ini :

```bash
Pernyataan 1 : a = 10
Pernyataan 2 : b = 20
```

Jika ditanya, apakah variabel `a` = `10` atau variabel `b` = `20`?

Tentu kita akan cek dulu kebenarannya

```bash
Pernyataan 1 : a = 10 // true
Pernyataan 2 : b = 20 // true
```

Apakah variabel `a` = `10` atau variabel `b` = `20`?

```bash
Pertnyataan 1 || Pernyataan 2 = true
```

**Contoh Program**

```java showLineNumbers
public class Main {
  public static void main(String[] args){
    byte a = 10;
    byte b = 20;

    System.out.println("a = 10? atau b = 20? : " + (a == 10 || b == 20)); // true

    System.out.println("a = 10? atau b = 5? : " + (a == 10 || b == 5)); // true

    System.out.println("a = 7? atau b = 20? : " + (a == 7 || b == 20)); // true

    System.out.println("a = 7? atau b = 5? : " + (a == 10 || b == 20)); // false
  }
}
```

**Penjelasan Program**

- Baris ke `6` bernilai `true` karena semua pernyataan benar **a = 10 atau b =20**
- Baris ke `8` bernilai `true` karena pernyataan pertama benar dimana `a` sama dengan `10`
- Baris ke `10` bernilai `true` kaerena pernyataan ke dua benar dimana `b` sama dengan `20`
- Baris ke `12` bernilai `false` karena semua pernyataan salah

</TabItem>
<TabItem value="Negasi" default>

Operator `Negasi` atau `kebalikan` berfungsi untuk membalik sebuah nilai `boolean`.

Jika suatu ekspresi `boolean` awalnya `true`, menggunakan operator `!` akan membuatnya menjadi `false`, dan sebaliknya.

Kita bisa menggunakan simbol `!` sebelum variabel seperti contoh `!nilaiBoolean`

**Contoh Program**

```java showLineNumbers
public class Main {
  public static void main(String[] args){
    boolean nilaiBoolean = true;

    // tanpa operator negasi
    System.out.println("Tanpa Operator Negasi = " + nilaiBoolean); // true

    // menggunakan operator negasi
    System.out.println("Menggunakan Operator Negasi = " + !nilaiBoolean); // false
  }
}
```

**Penjelasan Program**

- Baris ke `6` bernilai `true` karena tidak menggunakan operator negasi dimana program akan menampilkan isi dari `boolean` yang asli.
- Baris ke `9` bernilai `false` karena menggunakan operator negasi dimana yang nilai awalnya `true` akan di ubah ke `false`.
