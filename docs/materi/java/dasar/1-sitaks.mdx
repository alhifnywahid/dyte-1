Sintaks adalah sekumpulan aturan yang harus diikuti agar kode dapat dipahami dan dieksekusi oleh komputer. Setiap bahasa pemrograman memiliki sintaksnya sendiri, dan melanggar aturan sintaks akan mengakibatkan kesalahan (error) yang disebut kesalahan sintaks.

Java adalah bahasa pemerograman statis dimana setiap penulisan kode harus benar **contoh** di setiap akhir steatmen harus di akhiri dengan tanda titik koma `;`, dan serta tidak boleh mengubah sintaks sintaks seperti aslinya huruf besar di ubah ke huruf kecil.

## Program Halo Dunia

```java title="HaloDunia.java" showLineNumbers
public class HaloDunia {
  public static void main(String[] args){
    System.out.println("Halo Dunia");
  }
}
```

## Output

```bash
Halo Dunia
```
Di atas adalah contoh program sederhana menampilkan pesan `Halo Dunia` ke layar.

## Penjelasan

```java 
public class HaloDunia {

}
```

- `public class HaloDunia{ }` : ini adalah deklarasi class pada java, karena java adalah bahasa yang ber orientasi object maka setiap program harus di tuliskan di dalam class. di dalam class ini lah tempat membuat method

  - `public` : ini berarti class yang di buat bersifat public atau bisa di akses atau di panggil dari mana saja.

  - `class` : ini berarti mendefinisakn kalo anda menuliskan sebuah class bukan method.

  - `HaloDunia` : ini adalah nama class yang di buat, di java nama class dan nama file harus sama, berarti bisa kita ketahui program di atas filenya bernama *HaloDunia.java*

  - `{}` di tengah tengah tanda kurawal buka dan tutup inilah kita bisa menuliskan sebuah sintaks seperti *method*.


```java  
  public static void main(String[] args){

    System.out.println("Halo Dunia");

  }
```

- `public static void main(String[] args){ }` : ini adalah sekumpulan sintaks yang dinamakan method main, dimana method main inilah yang akan di jalankan oleh lebih awal oleh si program, penulisan method main atau method lainnya harus di dalam class.

  - `public` : sintaks ini berarti bahsa method tersebut bersifat public atau bisa di akses dari mana saja.

  - `static` : Ini adalah sintaks yang menunjukkan bahwa metode *main* adalah metode statis yang dapat dipanggil tanpa membuat objek dari kelas *HaloDunia*.

  - `void` : ini adalah tipe data kembalian method *main*, yang berarti method ini tidak mengembalikan nilai.
  - main : ini adalah method utama, seperti yang sudah di jelaskan di atas, method inilah yang akan di cari oleh program dan di eksekusi terlebih dahulu.

  - `String[] args` : ini adalah parameter yang di terima oleh method *main*, ini adalah daftar argumen yang dapat di teruskan ke program saat di jalankan.

  - `{ }` di tengah tengah tanda kurawal buka tutup ini lah kita bisa menuliskan sintaks sintaks lainnya. kita tidak bisa menuliskan sintaks *method* di dalam sebuah *method main* ataupun *method* lainnya

    - `System.out.println("Halo Dunia");` : sintaks inilah yang bertugas mencetak tulisan *"Halo Dunia"* ke layar. sintaks seperti inilah yang bisa kita tuliskan di dalam *method* main. kita harus mengakhiri sebuat sintaks dengan tanda titik koma.
