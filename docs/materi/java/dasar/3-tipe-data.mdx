Tipe data berfungsi untuk menentukan data pada sebuah variabel contoh bilangan bulat menggunakan tipe data `int`, bilangan pecahan menggunakan tipe data `float`.

## Alasan banyakknya tipe data di java

Salah satu tujuan banyaknya tipe di java adalah **Hemat Memory**, di mana programer java bisa menentukan tipe data yang ingin di gunakan pada variabel.

contoh si programer ingin membuat variabel yang bernilai `100` maka si programer bisa menggunakan tipe data `byte` karena byte adalah bilangan bulat dan maksimal bisa menampung nilai sebesar `127`.

Setiap tipe data di java mempunyai nilai minimum dan maksimum, silahkan simak penjelasan berikut untuk mengetahui ukuran-ukuran setiap tipe data.



## Tipe Data Primitiv

**Ciri - Ciri tipe data primitiv**

- **Representasi Nilai** : mempresentasikan nilai sebenarnya dan di simpan secara langsung di dalam memori. ini membuat operasi terhadap tipe data primitiv biasanya lebih cepat.

- **Ukuran Tetap** : memiliki ukuran yang tetap dan dan telah di definiskan sejak awal, misalnya `int` selalu memiliki ukuran `4 byte` dalam memori. dan `boolean` hanya bernilai `true` atau `false`.

- **Memiliki Nilai Default** : tipe data primitiv memiliki default meskipun kita tidak memberi nilai pada sebuah variabel misalnya `int` nilai defaultnya `0` dan `boolean` nilai defaultnya `true`

**Aturan penulisan tipe data primitiv**

di java cara penulisan tipe data primitiv ialah dengan huruf kecil, tidak boleh menggunakan huruf besar.

**Tipe data boolean**

Tipe data boolean di gunakan untuk mempresentasikan nilai benar atau salah, tipe data boolean hanya mempunyai 2 nilai yaitu : `true` : benar & `false` : salah.

- **Ukuran** : 1 bit
- **Nilai Default** : true
- **Deskripsi** : hanya meyimpan nilai `true` dan `false`

contoh penggunaan : 

```java title="Main.java"
public class Main {
  public static void main(String[] args){
    
    // highlight-start
    boolean peryantaanBenar = true;
    boolean pernyataanSalah = false;
    // highlight-end

  }
}
```

**Tipe data char**

Tipe data char di gunakan untuk mempresetasikan nilai ASCII, dimana tipe data char ini dipresentasikan dengan menggunakan simbol kutip tunggal (single quotes), seperti 'A', '7', atau '!'. 

- **Ukuran** : 2 bytes
- **Nilai Default** : ''
- **Deskripsi** : Menyimpan satu karakter / huruf / nilai ASCII.

contoh penggunaan : 

:::tip contoh benar

- contoh berikut benar karena masing masing variabel hanya berisi 1 karakter.

```java title="Main.java"
public class Main {
  public static void main(String[] args){
    
    // highlight-start
    char iniHuruf = 'A';
    char iniSimbol = '@';
    char iniAngka = '1';
    // highlight-end

  }
}
```
:::

:::danger contoh salah 

- variabel `iniHuruf` salah karena tipe data char hanya bisa untuk variabel yang menampung nilai dengan maksimal 1 karakter.

- variabel `iniSimbol` salah karena tidak menggunakan simbol (Single Quote) `'`

- variabel `iniAngka` salah karena menggunakan simbol (Dobule Quote) `"` bukan simbol (Single Quote) `'`

```java title="Main.java"
public class Main {
  public static void main(String[] args){
    
    // highlight-next-line-error
    char iniHuruf = 'Aku';
    // highlight-next-line-error
    char iniSimbol = @;
    // highlight-next-line-error
    char iniAngka = "1021";

  }
}
```

:::

**Tipe data byte**

Tipe data byte adalah tipe data integer yang di gunakan untuk mempresentasikan nilai bilangan bulat dengan maksimal nilai hingga 127.

- **Ukuran** : 1 byte
- **Nilai Default** : 0
- **Deskripsi** : untuk mempresentasikan bilangan bulat dari nilai minmal -128 hingga 127.

contoh penggunaan : 

:::tip contoh benar

- contoh berikut benar karena nilai yang di tampung oleh variabel `angka1` dan `angka2` tidak melewati nilai maksimal dari tipe data byte.

```java title="Main.java"
public class Main {
  public static void main(String[] args){
    
    // highlight-start
    byte angka1 = 100;
    byte angka2 = 127;
    // highlight-end

  }
}
```
:::

:::danger contoh salah

- variabel `angka3` salah karena nilai yang di tampung tersebut melebihi batas dari tipe data byte.

- variabel `angka4` salah karena menggunakan simbol titik `.` di tipe data byte tidak boleh menggunakan simbol atau huruf apapun selain angka.

```java title="Main.java"
public class Main {
  public static void main(String[] args){
    
    // highlight-next-line-error
    byte angka3 = 129;
    // highlight-next-line-error
    byte angka4 = 1.27;

  }
}
```

**Tipe data long**

Tipe data long adalah tipe data integer yang di gunakan untuk mempresentasikan nilai bilangan bulat dengan maksimal nilai hingga 9223372036854775807.

**Catatan** : khusus tipe data long jika nilainya lebih dari tipe data int makan harus menggunakan tambahan huruf `l` kecil atau `L` besar, `disarankan memakai huruf "L" besar`, jadi penambahan huruf `L` ini hanya untuk membedakan tipe data long dan int jika nilai dari variabel dengan tipe data long ini lebih dari nilai maksimum tipe data int.

- **Ukuran** : 8 bytes
- **Nilai Default** : 0
- **Deskripsi** : untuk meyimpan bilangan bulat dengan minimal nilai -9223372036854775808 sampai 9223372036854775807.

contoh penggunaan : 

:::tip contoh benar

- variabel `angka1` benar karena nilai yang di tampung tidak melebihi batas maksimal dari tipe data long.

- variabel `angka2` benar menggunakan tambahan huruf di akhir angka karena nilai tersebut sudah melebihi nilai maksimal dari tipe data int.

```java title="Main.java"
public class Main {
  public static void main(String[] args){
    
    // highlight-start
    long angka1 = 2147483647;
    long angka2 = 9223372036854775807L;
    // highlight-end

  }
}
```
:::

:::danger contoh salah

- variabel `angka3` salah karena nilai yang di tampung tersebut melebihi batas dari tipe data long.

- variabel `angka4` salah karena tidak menggunakan tambahan huruf "L" di setelah angka terakhir (nilai tersebut sudah melebihi dari maksimum tipe data int.)

- variabel `angka5` salah karena menggunakan simbol koma `,` di tipe data long tidak boleh menggunakan simbol atau huruf apapun selain `angka dan huruf (L) setelah angka terakhir` jika angka lebih dari maksimum tipe data int.

```java title="Main.java"
public class Main {
  public static void main(String[] args){
    
    // highlight-next-line-error
    long angka3 = 9223372036854775809L;
    // highlight-next-line-error
    long angka4 = 9223372036854775807;
    // highlight-next-line-error

}

**Tipe data float**

Tipe data float termasuk dalam tipe data floating point yang di gunakan untuk mempresentasikan nilai bilangan pecahan dengan maksimal nilai hingga 6 sampai 7 angka desimal.

di tipe data float anda harus menggunakan huruf `f` kecil atau `F` besar di akhir angka desimal jika ingin secara eksplisit mendeklarasikan angka tersebut sebagai tipe data `float`.

- **Ukuran** : 4 bytes
- **Nilai Default** : 0.0
- **Deskripsi** : untuk meyimpan bilangan pecahan dari 3.4e-038 sampai 3.4e + 038.

contoh penggunaan : 

:::tip contoh benar

    // highlight-end

  }
}
```
:::

:::danger contoh salah

- variabel `angka2` salah karena tidak menggunakan tambahan huruf `f` di akhir bilangan desimal.

## Tipe Data Non Primitiv

Untuk tipe data primitiv saya hanya akan menjelaskan bagian `String` saja, karena seperti `Array`, dll akan ada penjelasan tersendiri.

**Tipe Data String**

tipe data String digunakan untuk menyimpan kata atau kalimat baik berupa karakter huruf, angka dan simbol. 

Nilai dari tipe data String harus di apit oleh simbol (Double Qoute) `""`.

contoh penggunaan : 

:::tip contoh benar

- contoh berikut ini benar karena nilai variabel `nama` memnuhi aturan tipe data string `(menggunakan simbol double quote)`.

```java title="Main.java"
public class Main {
  public static void main(String[] args){
    
    // highlight-start
    String nama = "Alhifny Wahid";
    // highlight-end

  }
}
```
:::

:::danger contoh salah

- contoh pada variabel `nama1` salah karena nilainya di apit oleh simbol single quotes.

- contoh pada variabel `nama2` salah karena nilainya tidak di apit oleh simbol double quotes.

```java title="Main.java"
public class Main {
  public static void main(String[] args){
    
    // highlight-next-line-error
    String nama1 = 'Alhifny Wahid'
    // highlight-next-line-error
    String nama2 = Alhifny Wahid

  }
}
```
:::

## Kata Kunci (Var)

- Sejak versi Java 10, Java mendukung pembuatan varaibel dengan kata kunci var, sehingga kita tidak perlu menyebutkan tipe datanya.
- Namun perlu diingat, saat kita menggunakan kata kunci var untuk membuat variabel, kita harus menginisialisasi value/nila dari variabel tersebut secara langsung.

:::tip contoh benar

```java
public class Main {
  public static void main(String[] args) {
    var nama = "Indah Puspita";
    var alaamt = "Surabaya";
  }
}
```
- kodingan di atas tidak akan terjadi eror karena kita sudah menginisialisasi variabel langsung.

:::

:::danger contoh salah

```java
public class Main {
  public static void main(String[] args) {
    var nama;
    var alaamt;

    nama = "Indah Puspita";
    alamat = "Surabaya";
  }
}
```
- Kodingan di atas akan eror karena kita hanya melakukan deklarasi saja (Tidak langsung mengisi nilainya).

:::

## Kata Kunci final

- Secara default, variabel di Java nilainya bisa di ubah.
- Jika kita ingin membuat sebuah variabel yang datanya tidak boleh di ubah setelah pertama kali dibuat, kita bisa menggunakan kata kunci final sebelum penulisan tipe data.
- Istilah variabel seperti ini, banyak juga yang menyebutkan konstanta.

```java
public class Main {
  public static void main(String[] args) {
    final byte angka = 100;
  }
}
```

- nilai dari variabel byte sudah tidak bisa di ubah, jika kita mencoba memaksa untuk di ubah akan terjadi eror.

```java
public class Main {
  public static void main(String[] args) {
    final byte angka = 100;

    angka = 30; // ini akan terjadi eror
  }
}
```
