# Belajar BufferedReader

<hr />

**BufferedReader** adalah salah satu Class dalam Java yang digunakan untuk membaca data dari sumber input. Class ini terletak dalam paket `java.io`

Class ini tidak hanya untuk mengambil input dari keybord saja.

Class ini juga dapat di gunakan untuk membaca input dari file dan jaringan.

## Method BufferedReader

**BufferedReader** memiliki banyak sekali method untuk di gunakan, Berikut beberapa method yang umum di gunakan.

| **Method**      | **Deskripsi** |
|-----------------|-----------|
| readLine()      | Metode ini digunakan untuk membaca satu baris teks dari input. Ini akan membaca karakter sampai menemukan karakter baris baru ('\n') atau akhir aliran (EOF), kemudian mengembalikan baris tersebut sebagai String, termasuk karakter baris baru. | 
| close()         | untuk membaca nilai boolean | 
| mark()          | untuk membaca nilai byte |
| markSupported() | untuk membaca  nilai short |
| read()          | untuk membaca nilai int | 
| read(char[] cbuf, int off, int len) | untuk membaca nilai long | 
| ready()         | untuk membaca nilai float |
| reset()         | untuk membaca nilai double |
| skip(long)      | untuk membaca teks |

## Cara menggunakan BufferedReader

**BufferedReader** adalah Class dari paket **java.io** maka kita harus mengimport paketnya terlebih dahulu sebelum pembuatan class.

- mengimpor class **BufferedReader** saja.
```java
import java.io.BufferedReader;
```

- mengimpor semua class dari paket **java.io**
```java
import java.io.*;
```
## Contoh Penggunaan BufferedReader

Di sini saya akan mencontohkan penggunaan method **readLine()** saja untuk mendapatkan input dari user.

Untuk mendapatkan input dari user menggunakan **BufferedReader** ini kita membutuhkan class tambahan dari pustaka **java.io** juga yaitu :
```java
import java.io.IOException;
import java.io.InputStreamReader;
```

Silahkan copy source code berikut lalu ubah nama classnya dan jalankan program.

```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class Main {

    public static void main(String[] args) throws IOException {

        String nama;

        // Membuat objek inputstream
        InputStreamReader isr = new InputStreamReader(System.in);

        // membuat objek bufferreader
        BufferedReader br = new BufferedReader(isr);

        // Mengisi varibel nama dengan Bufferreader
        System.out.print("Inputkan nama: ");
        nama = br.readLine();

        // tampilkan output isi variabel nama
        System.out.println("Nama kamu adalah " + nama);

    }

}
```
