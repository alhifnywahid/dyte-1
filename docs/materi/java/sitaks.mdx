Sintaks adalah sekumpulan aturan yang harus diikuti agar kode dapat dipahami dan dieksekusi oleh komputer. Setiap bahasa pemrograman memiliki sintaksnya sendiri, dan melanggar aturan sintaks akan mengakibatkan kesalahan (error) yang disebut kesalahan sintaks.

Java adalah bahasa pemerograman static dimana setiap penulisan kode harus benar **contoh** di setiap akhir steatmen harus di akhiri dengan tanda titik koma `;`

## Source Code

```java title="HaloDunia.java" showLineNumbers
// highlight-next-line-error
public class HaloDunia {
  public static void main(String[] args){
    System.out.println("Halo Dunia");
  }
}
```

## Penjelasan

- `public class HaloDunia {` : definnisi awal dari sebuah kelas. Java adalah OOP dimana setiap progam harus berada di dalam sebuah kelas.
  - `public` : ini adalah kata kunci yang mengindikasikan bahwa kelas ini bisa di akses/di panggil  dari mana saja.
  - `class` : sintaks yang di gunakan untuk mengidefinikasn sebuah kelas.
  - `HaloDunia` : sintaks ini adalah nama dari sebuah kelasnya, dimana nama kelas harus sama dengan nama filenya, berarti nama file yang berisi kodingan ini adalah `HaloDunia.java`.
  `{` : Kurung kurawal buka menandakan awal dari blok kode yang termasuk dalam kelas, dimana setiap stea
