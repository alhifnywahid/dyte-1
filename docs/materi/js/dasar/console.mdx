## console.log()
```js
let nama = "Alhifny Wahid";
console.log(nama)
```

## console.table()
```js
let info = ["Java", "Javascript", "Golang", "PHP"];
console.table(info)
```

## console.dir()
```js
let info = {
  "nama": "Roki",
  "umur": 19,
  "alamat": "Surabaya",
};
console.dir(info)
```
