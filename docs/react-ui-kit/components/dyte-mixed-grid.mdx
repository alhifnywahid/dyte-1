# DyteMixedGrid

A grid component which handles screenshares, plugins and participants.

```jsx live
<DyteMixedGrid
  participants={[meeting.self]}
  pinnedParticipants={[meeting.self]}
  screenShareParticipants={[meeting.self]}
  plugins={[]}
  style={{ height: '360px', width: '100%' }}
/>
```
