# DyteNameTag

A component which shows a participant's name.

```jsx
<Row>
  <DyteNameTag participant={meeting.self} />
  {/* pass `meeting` to it to differentiate `you`. */}
  <DyteNameTag participant={meeting.self} meeting={meeting} />
</Row>
```
